### 진행방식

- 현재 repo를 개인 깃헙으로 fork 뜹니다.
- 주에 3번 진행하고 한 장을 2번에 나눠서 진행합니다.
  - 내용 공유(1일) + 블로그 정리 or 프로젝트 적용(1일)
- 정리한 내용을 스터디 시작 전까지 닉네임에 해당하는 branch에 PR을 올립니다.
- 발표
  - 번갈아가며 발표자와 질문자를 선정합니다.
  - 발표자는 조사한 내용에 대해 설명하고, 질문자는 이에 대해 궁금한 것을 질문합니다.
